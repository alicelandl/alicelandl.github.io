Hexo

Hexo是什么

Hexo是基于nodejs的一个快速、简洁且高效的博客框架。

本地安装hexo步骤

1、安装Git

2、安装node.js

3、安装hexo

Quick Start

  1. set up blog(在本地建立一个名为Hexo的文件夹，然后在此文件夹中右键打开Git Bash)

    Hexo随后会自动在目标文件夹建立网站所需要的文件，再运行npm install 安装node_modules.

  2. start the server (浏览器中打开http://localhost:4000/， Ctrl+C 停止Server)

  3. Create a new post

    刷新http://localhost:4000/， 可以发现已生成了一篇新文章 “My New Post”。

  4. Generate static files

    该命令执行完后，会在 D:\Hexo\public\ 目录下生成一系列html，css等文件。

  5. 编辑文章
    hexo new “My New Post”会在D:\Hexo\source_posts目录下生成一个markdown文件：My-New-Post.md

部署到Github

1、安装hexo关于git管理组件

2、配置_config.yml文件

3、deploy

部署完使用https://alicelandl.github.io/ 访问。

主题

  1. 默认主题是landscape， Hexo All Themes

  2. 主题更换
    1）下载安装next主题

    2）启用主题
    打开 站点配置文件， 找到 theme 字段，并将其值更改为 next
